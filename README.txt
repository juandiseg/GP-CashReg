DESCRIPTION
Through this app the workers/waiters of a restaurant can introduce to the ecosystem the information regarding the restaurant's daily functioning. Some of the functionalities it provides are providing a tool for workes to check-in/-out as well as tools to introduce sales and generate receipts.
------------------------------------------------------------------------------
GET RUNNING
In order for this app to work, please:
  - Add to the classpath all the .jar files from "imports".
  - Initiate a local instance of a database using the provided commands in: github.com/juandiseg/GP-AdminApp -> DBcommands.txt.
  - Change the values at "src/util/ManagerDB.java" in order to satisfy those of the instance of your local DB.
